## Installing The Bridge

<Callout type="warning" emoji="⚠️">
  For Developers Only
</Callout>

To be able to load the bridge you need to add the following to your fxmanifest.lua

```lua
lua54 'yes'

shared_scripts {
    'bridge/bridge.lua',
}

files {
    'bridge/**/**/client.lua',
    'bridge/**/*.lua',
}
```
